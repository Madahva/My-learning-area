# DOM Practice - JS  - SHOPPING LIST

It is a simple shopping list in vanilla Javascript. It allows users to add and remove shopping items

I have made use of DOM methods and properties like:

 - createElement()
 - textContent
 - remove()
 - appendChild()
 - classList()
